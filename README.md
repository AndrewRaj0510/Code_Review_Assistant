# Code Review Assistant

A modern web application built with **Next.js** and **Tailwind CSS** that helps developers review, analyze, and visualize code efficiently.

## Features
- Clean and responsive UI for seamless user experience
- Code visualization and review capabilities
- Dark and light mode support
- Component-based architecture for easy scalability

## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS
- **Styling:** Tailwind CSS for responsive and modern UI
- **Components:** Navbar, Background, CodeViewer, Headline, etc.

## Getting Started

### Prerequisites
- Node.js >= 18.x
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/code-review-assistant.git
   ```
2. Navigate to the project folder:
    ```bash
    cd code_review_assistant/frontend
    ```
3. Install dependencies(refer requirements.txt)
4. Start the development server:
    ```bash
    npm run dev
    ```
5. Open http://localhost:3000 in your browser.
